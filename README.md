# LinkMan

Private Online-Linkverwaltung mit einfacher Weboberflaeche.

## Features (MVP)
- Links anlegen, bearbeiten, loeschen
- Tags und Kategorien
- Suche und Filter
- Einfacher Login (privat)

## Projektstruktur
- `src/` App-Code
- `public/` statische Dateien
- `docs/` Projektnotizen

## Schnellstart
1. `npm install`
2. `npm run dev`
